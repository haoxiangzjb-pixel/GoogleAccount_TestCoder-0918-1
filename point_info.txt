Структура данных точки в Zig:

const Point = struct {
    x: f64,  // координата x (вещественное число двойной точности)
    y: f64,  // координата y (вещественное число двойной точности)
};

Пример экземпляра структуры:
const point = Point{ .x = 3.14, .y = 2.71 };

Данные сохраняются в бинарном формате в файл со случайным именем.
