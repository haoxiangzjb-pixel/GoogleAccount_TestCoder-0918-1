# Crystal Time Program

This directory contains a Crystal program that outputs the current time.

## Files
- `program_6546.cr` - The Crystal program that outputs the current UTC time

## Program Content
```crystal
# This program outputs the current time in Crystal
puts Time.utc
```

## How to Run
1. Install Crystal: `apt-get install crystal`
2. Run the program: `crystal run program_6546.cr`

Note: The program requires sufficient memory to compile and run.
